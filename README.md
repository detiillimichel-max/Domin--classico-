# 🀫 Jogo de Dominó

Um jogo de Dominó completo e interativo desenvolvido com HTML, CSS e JavaScript puro, com suporte a PWA para jogar offline.

## 🎮 Funcionalidades

- Jogue contra o computador
- Interface responsiva e moderna
- Sistema de pontuação
- Modo offline (PWA)
- Animações suaves
- Compatível com dispositivos móveis

## 🕹️ Como Jogar

1. Clique nas peças para jogá-las no tabuleiro
2. Combine os números das pontas para encaixar as peças
3. Use o botão "Passar Vez" quando não tiver jogadas
4. O jogo termina quando alguém atinge 5 pontos

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/domino-game.git

# Entre no diretório
cd domino-game

# Inicie um servidor local (Python)
python -m http.server 8000
